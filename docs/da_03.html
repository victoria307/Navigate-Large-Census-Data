Tigris is an R package that provides shapefiles and geographic boundary data down to the census tract and block group level. It‚Äôs especially useful for city-level analysis and any project involving geographic mapping.

üß©Example For example, now I want to load the map of blockgroups in the county of Blacksburg in 2022

#load the map, with the fips code of VA and Blacksburg
